# CockPY
public and open source version of the cbt2 ps im working on


installation steps:

- `pip install -r ./requirements.txt`

- `git clone https://github.com/lilmayofuksu/pyenet --recursive && cd pyenet`

- `python setup.py build`

- `python setup.py install`

To run CockPY:

`py -m cockpy`
